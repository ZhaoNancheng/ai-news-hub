## 技术架构

### 自动化流程
```
1. Cron 触发（每天 22:00）
   ↓
2. 调用 daily-ai-news-skill 获取新闻
   ↓
3. 生成 Markdown 文件
   ↓
4. 更新首页索引
   ↓
5. Git 提交并推送
   ↓
6. Vercel 自动部署
```

### 技术栈
- **VitePress** - 静态站点生成
- **Cron** - 定时任务
- **Git** - 版本控制
- **Vercel** - 自动部署
- **daily-ai-news-skill** - AI 新闻聚合
